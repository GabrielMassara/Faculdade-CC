# Colecionador de Pokémons - C

## Enunciado

> **Disclaimer:** O enunciado original desta questão foi fornecido em folha de papel. Por isso, este texto foi gerado com o auxílio de inteligência artificial para fins de documentação.

Durante sua jornada, você encontra diversos pokémons, mas alguns podem ser repetidos. Você precisa contar quantos pokémons únicos você já coletou e determinar quantos ainda faltam para completar sua coleção.

**Regras:**
- Existem exatamente 152 pokémons diferentes no total
- Pokémons com o mesmo nome são considerados iguais (não contam como pokémons diferentes)
- Você deve contar apenas pokémons únicos em sua coleção

**Entrada:**
- Primeira linha: um inteiro `n` representando o número de pokémons encontrados
- Próximas `n` linhas: o nome de cada pokémon encontrado (string)

**Saída:**
Uma linha no formato: `Falta(m) X pomekon(s).`, onde X é a quantidade de pokémons que ainda faltam para completar a coleção de 152.

## Exemplo

**Entrada:**
```
7
Charmander
Caterpie
Pidgeot
Rattata
Zubat
Zubat
Zubat
```

**Saída:**
```
Falta(m) 146 pomekon(s).
```
