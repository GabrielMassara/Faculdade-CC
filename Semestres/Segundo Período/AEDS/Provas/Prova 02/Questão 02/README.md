# Recreio Organizado - C

## Enunciado

> **Disclaimer:** O enunciado original desta questão foi fornecido em folha de papel. Por isso, este texto foi gerado com o auxílio de inteligência artificial para fins de documentação.

Durante o recreio da escola, os alunos de uma turma se organizam em fila para brincar. O professor observou que alguns alunos sempre ficam na mesma posição quando a fila é reorganizada em ordem crescente de altura.

Sua tarefa é determinar quantos alunos permanecerão na mesma posição quando a fila for reorganizada do **maior para o menor** (ordem decrescente).

### **Entrada:**
- A primeira linha contém um inteiro **n**, representando o número de casos de teste.
- Para cada caso de teste:
  - A primeira linha contém um inteiro **m** (1 ≤ m ≤ 1000), representando o número de alunos na fila.
  - A segunda linha contém **m** inteiros representando as alturas dos alunos na ordem atual da fila.

### **Saída:**
Para cada caso de teste, imprima uma linha contendo um único inteiro: o número de alunos que permanecerão na mesma posição quando a fila for reorganizada em ordem decrescente de altura.

### **Exemplo:**

**Entrada:**
```
3
3
100 80 90
4
100 120 30 50
4
100 90 30 25
```

**Saída:**
```
1
0
4
```
