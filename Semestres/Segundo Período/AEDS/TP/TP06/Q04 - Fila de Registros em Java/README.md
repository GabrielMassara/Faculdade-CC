# TP06Q04 - Fila de Registros em Java

## Enunciado

Crie uma **Fila de registros** baseada na fila de inteiros vista em sala de aula.  
Neste exercício, faremos **inserções (enfileirar)**, **remoções (desenfileirar)** e exibiremos os elementos da fila.





---




#### Estrutura e Conceito




A fila deve seguir o comportamento **FIFO (First In, First Out)** — o primeiro elemento inserido é o primeiro a ser removido.  
Cada elemento da fila deve armazenar uma **referência para um objeto da classe `Game`**.
