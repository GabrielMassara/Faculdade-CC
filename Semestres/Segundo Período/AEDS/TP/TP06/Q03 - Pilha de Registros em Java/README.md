# TP06Q03 - Pilha de Registros em Java

## Enunciado

Crie uma **Pilha de registros** baseada na pilha de inteiros vista em sala de aula.  
Neste exercício, faremos **inserções (empilhar)**, **remoções (desempilhar)** e exibiremos os elementos da pilha.






---






#### Estrutura e Conceito




A pilha deve ser **dinâmica**, implementada com **alocação flexível** (ponteiros).  
Cada elemento da pilha armazena um **registro `Game`** (ou um ponteiro para ele).  
As operações devem seguir a **lógica LIFO (Last In, First Out)**.
