# TP06Q02 - Lista com Alocação Flexível em C

## Enunciado

Refaça a questão anterior utilizando uma **lista dinâmica simplesmente encadeada** (alocação flexível).  
A lista deve armazenar **referências (ponteiros) para registros `Game`** e **replicar a interface** da versão sequencial, agora adaptada à estrutura encadeada.
