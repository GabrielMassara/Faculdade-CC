# AEDS - Trabalhos Práticos

Este repositório contém todas as atividades de **Trabalhos Práticos** da disciplina de **Algoritmos e Estruturas de Dados (AEDS)** do Segundo Período do curso de Ciência da Computação.

## Estrutura do Repositório

O repositório está organizado em trabalhos práticos (TP) numerados sequencialmente, cada um contendo diferentes questões sobre tópicos específicos de algoritmos e estruturas de dados.

### Lista de Trabalhos Práticos

#### **TP01** - Fundamentos
- **Q01**: Palíndromo (Java)
- **Q02**: Palíndromo Recursivo (C)
- **Q03**: Ciframento de César (Java)
- **Q04**: Alteração Aleatória (Java)

#### **TP02** - Manipulação de Strings e Lógica
- **Q01**: Álgebra Booleana (Java)
- **Q02**: Verificações "Is" (C)
- **Q03**: Inversão de String (Java)
- **Q04**: Soma de Dígitos (C)
- **Q05**: Verificação de Anagrama (Java)

#### **TP03** - Recursividade
- **Q01**: IS (C)
- **Q02**: Inversão de String (Java)
- **Q03**: Soma de Dígitos (Java)

#### **TP04** - Estruturas de Dados Básicas
- **Q01**: Classe em Java
- **Q02**: Struct em C

#### **TP05** - Algoritmos de Ordenação e Busca
- **Q01**: Pesquisa Binária (Java)
- **Q02**: Ordenação por Seleção (C)
- **Q03**: Heapsort (Java)
- **Q04**: Quicksort (C)
- **Q05**: Mergesort (Java)

#### **TP06** - Estruturas de Dados Lineares
- **Q01**: Lista com Alocação Sequencial (Java)
- **Q02**: Lista com Alocação Flexível (C)
- **Q03**: Pilha de Registros (Java)
- **Q04**: Fila de Registros (Java)

#### **TP07** - Árvores
- **Q01**: Árvore Binária (Java)
- **Q02**: Árvore Binária de Árvores (Java)
- **Q03**: Árvore AVL (C)
- **Q04**: Árvore Alvinegra (Java)

#### **TP08** - Tabelas Hash
- **Q01**: Hash Direta com Reserva (Java)
- **Q02**: Tabela Hash Direta com Rehash
- **Q03**: Hash Indireta

## Organização das Pastas

Cada questão possui sua própria pasta contendo:
- Código fonte (`.java` ou `.c`)
- `README.md` com descrição específica da questão
- Pasta `pubs/` com arquivos de testes
