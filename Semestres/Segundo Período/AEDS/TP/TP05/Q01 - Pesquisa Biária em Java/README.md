# TP05Q01 - Pesquisa Biária em Java

## Enunciado

Faça a inserção de alguns registros no final de um vetor e, em seguida, realize algumas pesquisas.  
A **chave primária de pesquisa** será o atributo **Name**.  



A entrada padrão é composta por duas partes:
1. A primeira é igual à entrada do **TP-04**.  
2. A segunda parte é composta por várias linhas, cada uma contendo um elemento que deve ser pesquisado no vetor.  
   A última linha terá a palavra **FIM**.  





A saída padrão será composta por várias linhas contendo as palavras **SIM** ou **NÃO**, indicando se cada um dos elementos pesquisados existe no vetor.  

Lembre-se de **ordenar os elementos** para a execução da pesquisa.  



Além disso, crie um **arquivo de log** na pasta corrente com o nome  
`suamatricula_binaria.txt`, contendo uma única linha com as seguintes informações:  

- Sua matrícula  
- Tempo de execução do algoritmo  
- Número de comparações  





Todas as informações do arquivo de log devem ser separadas por uma tabulação (`\t`).
