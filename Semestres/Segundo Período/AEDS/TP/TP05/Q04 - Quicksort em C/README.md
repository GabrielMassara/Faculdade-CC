# TP05Q04 - Quicksort em C

## Enunciado

Usando **vetores**, implemente o **algoritmo Quicksort** considerando que a **chave de ordenação** é o atributo **Release_date**.  
Em **caso de empate**, utilize como **critério secundário** o **AppID** do *Game*.





- **Entrada e saída padrão** são iguais às da questão de **Ordenação por Seleção**.  
- **Saída**: os **registros ordenados** pelo atributo **Release_date** (desempatando por **AppID**).





Além disso, crie um **arquivo de log** na pasta corrente com o nome `suamatricula_quicksort.txt`, contendo **uma única linha** com:





1. Sua **matrícula**  
2. Número de **comparações** (entre elementos do *array*)  
3. Número de **movimentações** (entre elementos do *array*)  
4. **Tempo de execução** do algoritmo de ordenação  





Todas as informações do arquivo de log devem ser separadas por uma tabulação (`\t`).
