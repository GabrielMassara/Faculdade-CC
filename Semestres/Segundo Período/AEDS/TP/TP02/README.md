# Trabalho Prático 02 - AEDS2

Este repositório contém as atividades do **Trabalho Prático 02** da disciplina de **Algoritmos e Estruturas de Dados 2 (AEDS2)**. Cada pasta corresponde a uma questão/atividade específica do TP.

## Estrutura Padrão de Cada Atividade

Cada pasta de atividade segue a mesma estrutura:

- **`README.md`**: contém o enunciado completo da atividade (copiado do Verde)
- **Arquivo de código fonte**: a última submissão (quando existir), baixada automaticamente do Verde
- **`pubs/` (pasta de testes públicos)**:
  - **`pubs.zip`**: arquivo zip com os testes públicos baixados do Verde
