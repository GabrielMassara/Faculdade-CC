# TP02Q01 - Algebra Booleana

## Enunciado

Crie um método **iterativo** que recebe uma string contendo uma expressão booleana e o valor de suas entradas e retorna um booleano indicando se a expressão é verdadeira ou falsa. 





Cada string de entrada é composta por um número inteiro ``n`` indicando o número de entradas da expressão booleana corrente. Em seguida, a string contém ``n`` valores binários (um para cada entrada) e a expressão booleana. 





Na saída padrão, para cada linha de entrada, escreva uma linha de saída com SIM / NÃO indicando se a expressão corrente é verdadeira ou falsa.
