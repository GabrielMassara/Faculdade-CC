# Laboratório 04 - AEDS

Este repositório contém as atividades do Laboratório 04 da disciplina de Algoritmos e Estruturas de Dados 2 (AEDS2). Cada pasta corresponde a uma atividade específica do laboratório.

## Estrutura Padrão de Cada Atividade

Cada pasta de atividade segue a mesma estrutura:

- **`README.md`**: Contém o enunciado completo da atividade, explicando o problema a ser resolvido e os requisitos
- **Arquivo de código fonte**: O arquivo principal com a implementação da solução
- **`pubs/` (pasta de testes públicos)**:
  - **`pub.in`**: Arquivo com os casos de teste de entrada
  - **`pub.out`**: Arquivo com as saídas esperadas correspondentes às entradas
